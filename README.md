# AmazonDashButtonEsp32Server
This project uses an ESP8266/ESP32 to create a Server for the famous Amazon Dash Button.


## Hardware
* Amazon Dashbutton
* ESP Development board or equivalent


## Tested with
* Amazon Dash Button v2
* ESP01
